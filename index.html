Sobre mí

Perfil técnico

Quién soy

Soy estudiante de Desarrollo Front-End con experiencia práctica en la creación de interfaces web, aplicaciones SPA, componentes reutilizables, formularios, consumo de APIs y organización de proyectos con Git y GitHub.

Este portafolio reúne los trabajos realizados durante el bootcamp y presenta mi evolución técnica de forma clara, visual y ordenada.

Qué busco proyectar

Busco mostrar una identidad profesional enfocada en el desarrollo web moderno, con capacidad de aprender nuevas herramientas, documentar proyectos y transformar ejercicios del curso en evidencia concreta para oportunidades laborales en TI.
