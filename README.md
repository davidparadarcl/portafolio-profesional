# Portafolio Profesional - David Parada

Portafolio personal desarrollado para la Actividad N° 3 del Módulo Portafolio Profesional.

## Objetivo

Presentar proyectos, tecnologías y aprendizajes desarrollados durante el bootcamp de Desarrollo Front-End, aplicando buenas prácticas de organización, diseño responsive, documentación y difusión digital.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript ES6+
- SCSS / SASS como tecnología trabajada en el curso
- Bootstrap 5 como herramienta trabajada en el curso
- Vue 3 como framework principal trabajado en proyectos
- Vue Router
- Vuex
- Axios
- Git y GitHub
- GitHub Pages para publicación
- Node.js y npm como entorno de desarrollo

## Secciones del portafolio

- Inicio
- Sobre mí
- Stack tecnológico
- Proyectos desarrollados
- Ruta Aprendizaje
- Contacto

## Proyectos presentados

- Weather SPA Vue
- Weather Front-End M2 / M3
- Mini SPA Vue Router
- Registro Vue Formulario
- Manejo de Eventos Vue
- LoginForm + pruebas unitarias
- Modelo de cajas
- Meteoro
